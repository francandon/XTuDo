# Downloading NuSTAR archive data

In order to download the publicly available data from the NASA missions you have to acess the HEASARC archive webpage:

[HEASARC archive webpage](https://heasarc.gsfc.nasa.gov/cgi-bin/W3Browse/w3browse.pl)

Type the sourrce name in **Object Name or Coordinates**, for example Sgr A*. 
NuSTAR. 


Downloading them is as simple as clickling the white box for the observations you want to download.
